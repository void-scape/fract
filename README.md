# Fract

Hardware accelerated viewer of the [Mandelbrot set](https://en.wikipedia.org/wiki/Mandelbrot_set#Computer_drawings).

Utilizes perturbation theory to dramatically increase zoom depth. I highly recommend reading this [blog post](https://philthompson.me/2022/Perturbation-Theory-and-the-Mandelbrot-set.html) to learn more.
